# Data-Visualization-Projects
Data Visualization Dashboards 
This portfolio showcases a variety of data visualization projects focused on transforming data into clear, impactful dashboards.

Each project highlights the ability to present insights in a simple, structured, and visually engaging way to support understanding and decision-making.

The dashboards are built using tools such as Power BI, Google Looker Studio, and Microsoft Fabric.

This work reflects a strong focus on clarity, storytelling, and designing visuals that make data easy to interpret and act on.
